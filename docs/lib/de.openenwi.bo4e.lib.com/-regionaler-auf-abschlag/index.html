RegionalerAufAbschlag

public final class RegionalerAufAbschlag
Content copied to clipboard

Mit dieser Komponente können Auf- und Abschläge verschiedener Typen im Zusammenhang mit regionalen Gültigkeiten abgebildet werden.

Hier sind auch die Auswirkungen auf verschiedene Tarifparameter modelliert, die sich durch die Auswahl eines Auf- oder Abschlags ergeben.
